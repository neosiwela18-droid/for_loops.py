# for_loops.py
What are for loops: 
A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

**What this code does:**
The code creates two lists of names with mixed capitalization, then prompts the user to input two additional names. It loops through the user-entered names first, printing a birthday invitation to each one with proper capitalization using `.title()`. Then it combines both original name lists into a set (removing duplicates) and loops through them again, printing the same invitation message to each person.
